Limitless

Adds options to customize how many skill points you get, how many points that can be put into skills, and how many points class mods add to skills.

Version 1.5:
  - Added a 5 skill points per level option
  - Added an option to make you need to spend 50 points to go down in the skill trees.

Version 1.4:
  - Fixed Salvador's Yippee Ki Yay skill having excess points in the 10 point skills option.
  - Added options for class mod scaling, about 2x the skill points and 10x the skill points.
    
Version 1.3:
  - Bugfix: Salvador's Yippee Ki Yay skill wasn't affected.
  - Class mods now also scale with the increased skill points.
  - Fixed Silence the Voices self hits scaling up being enabled by default.
    
Version 1.2:
  - Added options to change skill points per level: 1, 2, and 10.
  - Added options to change skills max points: 10, and 100.
  - Added option for Silence the Voices self hit chance to increase.
    
Version 1.1:
  - Bugfix: Some of Maya and Salvador's skills didn't get their max points to be 100.
    
Version 1.0
  - Initial release.
